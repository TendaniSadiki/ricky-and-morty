Rick and Morty Explorer

Explore the universe of Rick and Morty with this fun and interactive website. Dive into detailed character profiles, iconic locations, and memorable episodes from the hit show.

Features

  • Characters: Browse and discover characters from the show, complete with detailed information and images.
  • Locations: Explore iconic locations across dimensions, including their type and associated dimensions.
  • Episodes: Relive the best moments from the show with a list of episodes, their air dates, and summaries.

Technologies Used

  • React: For building the user interface.
  • Create React App: To set up the project quickly.
  • Axios: For fetching data from the Rick and Morty API.
  • CSS Modules: For component-level styling.

Getting Started

To get a local copy up and running, follow these steps:

Prerequisites

  • Node.js installed on your machine.
  • npm (Node Package Manager).

Installation

  1. Clone the repository:
    git clone https://github.com/TendaniSadiki/ricky-and-morty.git
  2. Navigate to the project directory:
    cd rick-and-morty
  3. Install dependencies:
    npm install
  4. Start the development server:
    npm start

    The app will open in your default browser at http://localhost:3000.
